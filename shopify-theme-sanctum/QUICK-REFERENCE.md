# Quick Reference - Theme Customization

## 🎨 Most Used Customizations

### Change Brand Colors (5 minutes)
1. Customize → Theme Settings → Colors
2. Set **Accent Color** (default #c5a880)
3. Set **Background Color** (default #090807)
4. Set **Text Color** (default #f5f3ef)
5. Save

### Adjust Cinematic Effects (3 minutes)
1. Customize → Theme Settings → Cinematic Environment
2. **Fog Opacity**: 0-100% (or toggle off)
3. **Light Sweep Opacity**: 0-100% (or toggle off)
4. **Motion Intensity**: 1-10 slider
5. Save

### Modify Hero Section (2 minutes)
1. Customize → Hero Slideshow
2. Edit **Slide Title** and **Subtitle**
3. Change button URLs and text
4. Toggle **Auto Rotate** or adjust **Slide Duration**
5. Save

### Add Newsletter Section (1 minute)
1. Customize → Newsletter
2. Edit **Title**, **Description**, button text
3. Save (form automatically connects to store email)

### Customize Typography (5 minutes)
1. Customize → Theme Settings → Typography
2. Choose **Heading Font** (serif/sans)
3. Choose **Body Font**
4. Adjust **Base Font Size** (12-20px)
5. Adjust **Letter Spacing** for uppercase
6. Save

## 🎯 Common Tasks

### Change Announcement Text
Customize → Header → **Announcement text**

### Adjust Mobile Layout
Customize → Theme Settings → Layout → **Max Width** (reduce to ~960px)

### Disable Animations
Customize → Theme Settings → Animations
- Uncheck **Enable Scroll Animations**
- Uncheck **Enable Hover Effects**
- Disable **Particle System**

### Add Custom Logo
Customize → Theme Settings → **Logo Image** (upload PNG/JPG)

### Change Newsletter CTA
Customize → Newsletter
- Title, description, button text all editable

### Show/Hide Components
Customize → Theme Settings → Components
- Toggle **Show Breadcrumbs**
- Toggle **Show Product Reviews**
- Toggle **Show Related Products**

## 🔥 Pro Tips

### For Mobile Optimization
1. Set **Max Width** to 960px (from 1280px)
2. Reduce **Particle Count** to 20-30
3. Disable **Fog** and **Light Sweep** on mobile (automatic)
4. Test on actual mobile device

### For Dark Mode / High Contrast
1. Background Color → #000000
2. Text Color → #FFFFFF
3. Accent Color → #FFFF00
4. Disable Grain Texture

### For Professional / Minimal Look
1. Disable all Cinematic effects
2. Set **Border Radius** to 8px (some rounding)
3. Reduce **Motion Intensity** to 3-4
4. Use neutral colors (grays, blacks)

### For Bold / Vibrant
1. Bright **Accent Color** (e.g., #FF00FF)
2. High **Motion Intensity** (8-10)
3. Enable all effects
4. Use **Letter Spacing** 3-4

## ❓ FAQ

**Q: Where do I change the homepage content?**
A: Customize → Select each section (Hero, Newsletter, etc.) → Edit

**Q: How do I add a new section?**
A: Customize → + Add Section → Choose from list

**Q: Can I have different colors per section?**
A: Yes! Some sections have their own color overrides

**Q: Will animations break on older phones?**
A: No, theme auto-detects and disables on slow devices

**Q: How do I undo changes?**
A: Customize → Revert to previous version (Shopify admin)

**Q: Can I add my own sections?**
A: Yes! Edit Liquid template files in the theme

**Q: Why are images blurry?**
A: Upload high-quality images (2000x2000px+)

**Q: How do I speed up the site?**
A: Enable lazy loading + responsive images in theme settings
